## Key Features Documentation

### Events System

The events system consists of:
- `/web/_data/events.yml` - Event data
- `/web/_data/event_locations.yml` - Geographic coordinates for events
- `/web/events.ics` - Auto-generated iCalendar file
- `/web/events.json` - Auto-generated JSON for event maps

### Schedule System

The show schedule system consists of:
- `/web/_data/schedule.yml` - Show schedule data
- `/web/schedule.md` - Schedule display page with calendar
- `/web/schedule.ics` - Auto-generated iCalendar file for shows

### Configuration

Global timezone setting: `/web/_config.yml` → `site_timezone`

## Making Changes

All ICS and JSON files are auto-generated by Jekyll from YAML data files. To make changes:

1. Edit the YAML data files in `/web/_data/`
2. Rebuild Jekyll: `cd stacks && docker-compose restart`
3. Files are automatically regenerated

Never edit the ICS or JSON files directly - they will be overwritten on next build.
