# Overview 

## What is Cubie-3?

Cubie-3 is a simple 3d printable cube robot you can build yourself. It's a flexible design that can be used for a variety of applications and is powered by a Raspberry Pi 5 with the Sunfounder Fusion HAT+.

## Features

- A front facing Raspberry Pi Camera Module v3 for computer vision projects.

- Four mecanum wheels for omnidirectional movement.

- A modular design that allows for easy customization and expansion.

- Raspberry Pi 5 powered for high performance.

- Sunfounder Fusion HAT+ for 
    - Motor control
    - Battery power
    - Microphone for sound input and speech recognition
    - Speaker for sound output and speech synthesis

- Perfect for learning robotics, [ROS/ROS2](/learn/learn_ros), computer vision, voice control and more!
