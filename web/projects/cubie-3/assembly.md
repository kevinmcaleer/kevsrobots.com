## Assembling the Cubie-3

### Preparing the Pillars

The 4 pillars have captive nut slots that allow an M3 nut to be inserted. To make a good fit, you can use the tip of a soldering iron to heat the nut and push it into the slot.

{% include gallery.html images="assets/img/heatset.png" titles="Inserting Captive Nuts into Pillars" cols="2" noborder=true %}

---

### Start with the base

1. Add the Pi to the base using 4x M2.5 screws.
1. Attach the Fusion HAT+ on top of the Pi using 4x M2.5 standoffs and gpio extender.
1. Connect the camera cable to the Pi and Camera Module v3.
